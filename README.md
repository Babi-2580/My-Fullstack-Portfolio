🎯 Project Overview
A professional portfolio website showcasing my skills and projects as a Full-Stack Developer. Features a sleek dark theme with blue accents, complete admin panel for project management, and contact form.

✨ Features
✅ Modern dark theme with blue accents (#0B1120)

✅ Complete CRUD operations for projects

🚀 Quick Start
bash
# Clone repository
git clone https://github.com/Babi-2580/my-portfolio.git
cd my-portfolio

# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install

# Setup database (MySQL)
# Create database 'portfolio_db' and run the SQL in the docs folder

# Start backend (port 3001)
cd ../backend
npm start

# Start frontend (port 3002)
cd ../frontend
npm start
